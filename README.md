## Features  

- View all posts on the main page  
- Create an account and log in  
- Edit your profile and delete your account  
- Visit user profiles to see their info and posts  
- Like posts and write comments  

## Technologies  

### Frontend  
- React  
- Redux  
- Axios  

### Backend  
- Express (Node.js)  
- Prisma ORM  
- JWT token  

### Database  
- PostgreSQL  

### Picture storage  
- Firebase
